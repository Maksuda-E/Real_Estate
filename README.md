# Real Estate Price Prediction Project

This project predicts the price of a property based on its features.

# Files

main.py is used for training and evaluation.

app.py is used for the Streamlit deployment.

src folder contains modular Python files for data loading, preprocessing, training, and prediction.

# How to run

Install dependencies:

pip install -r requirements.txt

Run training:

python main.py

Run the Streamlit app:

streamlit run app.py
Dataset

Place the dataset file inside:

data/final.csv

Output

The app takes property details as input and returns the predicted house price.
